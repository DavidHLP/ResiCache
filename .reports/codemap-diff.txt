ARCHITECTURE SUMMARY
================================================================================
The project implements a Spring Cache abstraction over Redis with protections
against common cache failure modes:

  Cache Penetration  → BloomFilterHandler + NullValueHandler
  Cache Breakdown    → SyncLockHandler (distributed lock)
  Cache Avalanche    → TtlHandler (TTL variation)
  Hot Keys           → PreRefreshHandler (async refresh)

Chain of Responsibility pattern used for handler execution with support for
post-processing callbacks.
