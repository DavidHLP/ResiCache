## Implementation Notes

### DOC-02 Actuator 端点设计
端点需暴露以下指标:
- cache.hits: 缓存命中次数
- cache.misses: 缓存未命中次数
- cache.hitRate: 缓存命中率
- bloomFilter.enabled: 布隆过滤器启用状态
- bloomFilter.expectedInsertions: 预期插入数
- bloomFilter.falseProbability: 误判率
- preRefresh.enabled: 预刷新启用状态
- preRefresh.poolSize: 预刷新线程池大小

### DOC-03 事件类型设计
事件类需提供以下信息:
- CacheHitEvent: cacheName, key, timestamp, ttl
- CacheMissEvent: cacheName, key, timestamp
- PreRefreshTriggeredEvent: cacheName, key, timestamp, reason
- TtlExpiryEvent: cacheName, key, timestamp, expiredTtl
